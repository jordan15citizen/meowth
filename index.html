Meowth

Meowth is a tool to make git simpler. It doesn't want to be flashy, but is very helpful. Written in Nim, it is fast and can be used in Termux and Linux distros through selecting the --os flag in the Nim compiler process.

Using Meowth is very easy. It shortens git commands. Meowth uses ANSI colours which makes your git commands easily debuggable.

Building Meowth

Building is very easy thanks to Nimble. The command to build Meowth is: 

nimble releaseBuild
  • This will make tiny binaries
  • The compiled binary will be fast
  • Debugging symbols are stripped out
