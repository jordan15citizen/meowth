# Meowth
Obviously, the name is a reference to **Meowth**, a **Pokémon**.
It was created so you can quickly **push changes**. Being written in **Nim**, it is **blazing fast**!
There is a **payday reference** somewhere, you must find it out yourself.
# Build
Build instructions:
```bash
sudo apt install nim
```
Or, if you are in **Termux**:
```bash
apt install nim
```
Clone the URL with GitHub CLI or git. then:
* Move into the **cloned folder**
* Build with:
```bash
nimble releaseBuild

# Usage
You can use **Meowth** to **easily** push to GitHub.
You can **watch** GitHub Actions and **sync** forks.
It has **shortcuts** for most **command**, being very **modular**
